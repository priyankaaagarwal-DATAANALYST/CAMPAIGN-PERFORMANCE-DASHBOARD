# 🎯 Campaign Performance Analytics Dashboard

## 📊 Project Overview

An executive marketing intelligence dashboard designed to maximize advertising utility and budget efficiency across audio, podcast, and digital campaigns. This interface provides an aggregated view of key performance indicators (KPIs) to help media buyers instantly identify high-yielding creatives and eliminate underperforming placements.

---

## 🛠️ Tech Stack

* 📊 **Power BI Desktop** – Core engine used for report layout, data compilation, and interface design.
* 🎛️ **Advanced Bookmark Navigation** – Utilized to build a seamless multi-view environment, optimizing space and rendering speed.
* 🧠 **DAX Time Intelligence** – Structured to calculate week-over-week (WoW) performance deltas dynamically.

---

## 💡 Features & Highlights

### • Walkthrough of Key Visuals

#### 🧭 Multi-View Sidebar Navigation Panel

* Features an integrated, high-utility sidebar menu that lets stakeholders toggle between specialized granular views: **Creative**, **Tactics**, **Placement**, **Partners**, and **Unit Length**.

#### 📈 Core Efficiency Metrics (Top Panel)

* **CTR %:** 28% (with a dynamic tracking notification of $-1899.2\%$ vs last week).
* **CPC This Week:** 26.9% (tracking an increase of $+26.6\%$ vs last week).
* **VCR %:** 44.8% (showing an optimization increase of $+2.7\%$ vs last week).

#### 📝 Partner Performance Matrix (Central Panel)

* A comprehensive breakdown table by **Creative** and **Partner** (e.g., ACAST, Amazon, iHeart, Spotify) displaying real-time metrics including **CTR%**, **CTR $\Delta$ %**, **CPC%**, **CPC $\Delta$ %**, **VCR %**, and **VCR $\Delta$ %**.

#### 📉 Trend Analysis Time Series (Right Panel)

* **CTR (%) Time series:** Tracks long-term click-through rate fluctuations from April 2025 through June 2025.
* **CPC Time series:** Displays cost-per-click progression over the same period.
* **Audio Completion Rate (%) Time series:** Monitors content engagement longevity, featuring an interactive data point tooltip (e.g., *11 June 2025: Audio Completion Rate(%) 42.3%*) with drill-down capabilities.

### • Business Impact & Insights

* **Budget Optimization:** Allows immediate identification of partners with high VCR % but low CTR %, indicating strong content engagement but weak call-to-action performance.
* **Real-Time Agility:** Week-over-week delta metrics give media buyers the visibility needed to pause underperforming creative variations before budget waste scales.
* **Granular Auditing:** The modular navigation structure splits massive campaign data into digestible dimensions (Tactics, Placement, Unit Length), speeding up executive review cycles.
